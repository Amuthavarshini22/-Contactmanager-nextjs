The Contact Manager is a full-stack web application built using Next.js that allows users to securely manage their contacts. The application includes authentication and complete CRUD (Create, Read, Update, Delete) functionality.This project demonstrates modern full-stack development using the Next.js App Router, Server Actions, and API handling.

Features
  User Authentication (Login/Logout)
  Add new contacts
  Edit existing contacts
  Delete contacts
  View all contacts
  Real-time UI updates
  Responsive and clean UI
  Structured folder architecture

Tech Stack
  Frontend: Next.js (App Router), React
  Backend: Next.js Server Actions & API Routes
  Language: TypeScript
  State Management: React Hooks
  Styling: Global CSS
  Data Handling: JSON-based local storage
  Version Control: Git & GitHub
